# Optimización de Corte en Industria Textil 🧵

Este proyecto utiliza **Python** y la librería **PuLP** para resolver el problema de *Trim Loss* (desperdicio de retales). 

## Objetivo
Maximizar el aprovechamiento de rollos de tela de 10 metros para producir camisas y pantalones, reduciendo el residuo textil al mínimo posible.

## Requisitos
- Python 3.x
- PuLP (`pip install pulp`)
